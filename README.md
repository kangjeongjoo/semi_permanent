# 반영구 눈썹·입술 시뮬레이터 (개인용)

내 정면 사진에 반영구 눈썹·입술 디자인을 미리 입혀보고, AI 추천을 받아 비교·저장하는 개인용 웹앱입니다. **사진과 모든 처리는 브라우저(기기) 안에서만 이뤄지며 서버로 전송되지 않습니다.**

## 실행 방법

```bash
npm install      # 최초 1회
npm run dev      # 개발 서버 (http://localhost:3000)
# 또는 배포용 빌드
npm run build && npm start
```

브라우저에서 http://localhost:3000 접속 → 정면 사진 업로드.

> 얼굴 인식 모델(MediaPipe)과 WASM은 최초 1회 CDN에서 내려받습니다. 다운로드 후 사진 분석은 기기 내에서 실행되며, 사진 자체는 외부로 전송되지 않습니다. 첫 실행 시 인터넷 연결이 필요합니다.

## 사용 흐름

1. **정면 사진 업로드/촬영** (무표정·밝은 조명·앞머리 올림 권장)
2. **자동 얼굴 인식 + AI 추천** — 얼굴형을 분석해 어울리는 눈썹/입술을 이유와 함께 자동 적용
3. **직접 수정** — 눈썹(기법·모양·색·굵기·아치·꼬리 등)과 입술(기법·색 계열·채도·발색 등)을 자유롭게 조정
4. **Before / After 비교** — 슬라이더로 원본 ↔ 합성 비교
5. **시안 저장 / 다운로드** — 마음에 든 시안을 기기에 저장(IndexedDB)하거나 이미지로 내려받기

## 구조

| 파일 | 역할 |
| --- | --- |
| `lib/faceLandmarker.ts` | MediaPipe 얼굴 랜드마크 검출(기기 내) |
| `lib/recommend.ts` | 규칙 기반 AI 추천 (얼굴 특징 → 디자인 매핑) |
| `lib/compositor.ts` | Canvas 합성 — 눈썹 파라메트릭 생성 + 입술 색 블렌딩 |
| `lib/designLibrary.ts` | 눈썹·입술 유형 라이브러리 (템플릿 데이터) |
| `lib/storage.ts` | 시안 저장소 (IndexedDB) |
| `app/page.tsx` | 메인 화면 (업로드·추천·편집·비교·저장) |
| `components/` | 컨트롤 패널·비교 슬라이더·갤러리 UI |

## 유형 추가하기

`lib/designLibrary.ts`의 `BROW_TEMPLATES` / `LIP_TEMPLATES` 배열에 항목을 추가하면 바로 목록에 노출됩니다. 색 계열은 `BROW_COLORS` / `LIP_COLOR_FAMILIES`에서 관리합니다.

## 참고

합성 결과는 **상담·참고용 미리보기**이며 실제 시술 결과와 다를 수 있습니다. 사진은 정면일수록 인식·합성 정확도가 높습니다.
